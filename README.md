Atividade Prática: Desenvolvimento de CRUD e Aplicação de Testes
Objetivo: Desenvolver um pequeno sistema que contemple as quatro operações básicas (CRUD - Create, Read, Update, Delete) e validar seu funcionamento através de técnicas de testes de software.
Requisitos:
Implementar o sistema funcional (tema livre).
Aplicar, no mínimo, dois tipos de testes (ex: Testes Unitários, Testes de Integração, Caixa-Preta, etc.).
Entrega:
Código Fonte: O sistema desenvolvido (link do repositório ou arquivo .zip)
Uma breve explicação de quais testes são mais conhecidos e porque selecionou os que você fez.
Relatório Técnico: Documento descrevendo quais testes foram escolhidos, como foram implementados e os resultados obtidos (prints ou logs de execução).
